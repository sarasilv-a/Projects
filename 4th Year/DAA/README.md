## Virtual Environment Setup

The following instructions are optional and you can choose between other tools to manage your Python environment.

If you using _venv_, follow these steps to set up your virtual environment (change the Python version if necessary):

- Create virtual environment:
```bash
python3.13 -m venv <venv name>
```

- Activate the virtual environment:
```bash
source <venv name>/bin/activate
```

- Installation of the needed packages:
```bash
pip install -r requirements.txt
```

- After finishing, you can deactivate the virtual environment: 
```bash
deactivate
```
