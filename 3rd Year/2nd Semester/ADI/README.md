## Project: Productivity and Credit Risk Analysis
---

**Learning Outcomes**

- Apply the CRISP-DM methodology for data mining and analysis  
- Perform data preprocessing, including handling missing values, inconsistencies, and outliers  
- Conduct exploratory data analysis to identify patterns and correlations  
- Develop and evaluate machine learning models for regression and classification tasks  
- Transform regression problems into classification tasks for enhanced analysis  

---

## Project Overview

This project, developed for the *Aprendizagem e Decisão Inteligentes* course at Universidade do Minho, involved analyzing two datasets to extract insights using machine learning techniques. The first dataset, selected by the group, focused on worker productivity in garment factories, while the second dataset analyzed credit risk for loan approvals. The goal was to understand factors affecting productivity and creditworthiness, applying the CRISP-DM methodology.

Key functionalities include:  

- Data cleaning and preprocessing to handle missing values, inconsistencies, and outliers  
- Exploratory data analysis to identify correlations and critical variables  
- Transformation of regression problems into classification tasks for improved modeling  
- Application of machine learning models to predict productivity and credit risk  
- Evaluation of model performance using appropriate metrics  

### Main Features:
- CRISP-DM methodology for structured data analysis (Business Understanding, Data Understanding, Data Preparation, Modeling, Evaluation)  
- Handling of missing values, invalid data, and outliers in both datasets  
- Correlation analysis to identify key variables impacting productivity and credit risk  
- Use of machine learning models (e.g., regression, classification) for predictive analysis  
- Visualization of data insights through box plots, correlation matrices, and statistical tables  

---

## Implementation

- **Tools:** Python with libraries like `pandas`, `numpy`, `scikit-learn`, and `matplotlib` for data analysis and visualization  
- **Dataset 1 (Productivity):** Analyzed garment factory data with attributes like date, department, team, targeted productivity, overtime, incentives, and actual productivity  
- **Dataset 2 (Credit Risk):** Analyzed loan approval data with attributes like person age, income, education, loan amount, and credit history length  
- **Preprocessing:** Corrected inconsistencies (e.g., invalid quarter values, negative productivity), removed low-utility attributes, and handled outliers using statistical methods  
- **Modeling:** Applied regression and classification models; transformed regression tasks into classification for enhanced interpretability  

---

## Testing & Results

- Validated data preprocessing steps, ensuring no missing values or inconsistencies remained  
- Correlation analysis revealed key variables like overtime and incentives for productivity, and income and loan amount for credit risk  
- Model performance evaluated using metrics like accuracy, precision, recall, and mean squared error  
- Classification models showed improved interpretability after transforming regression tasks  
- Visualizations (e.g., box plots, correlation matrices) confirmed data insights and model effectiveness
