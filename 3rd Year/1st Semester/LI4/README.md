## Project Overview

This project involved the development of **Montagem Belas Pizzas**, a web-based application designed for the *Laboratórios de Informática IV* course at Universidade do Minho. The system supports the production line of Belas Pizzas, a company transitioning from a family restaurant to a frozen pizza factory, by providing an interactive platform for managing pizza assembly, inventory, and employee operations.

Key functionalities include:  

- User authentication and role-based access (administrators and employees)  
- Management of products, ingredients, purchases, and sales  
- Real-time inventory tracking and recipe updates  
- Interactive interface for guiding pizza assembly processes  
- Employee management with performance statistics and profiles  

### Main Features:
- 3-tier architecture (Data Access, Business, UI) for clear separation of responsibilities  
- Database management with SQLDataAccess for efficient data operations  
- Interactive UI developed with Blazor Server, styled using Tailwind CSS and Bootstrap  
- Comprehensive use case diagrams and domain modeling for system clarity  
- Support for scalability, accessibility, and sustainability by reducing paper-based manuals  

---

## Implementation

- **Languages and Frameworks:** C# with Blazor Server for the UI, SQL for database operations, Tailwind CSS and Bootstrap for styling  
- **Architecture:** 3-tier architecture with Data Access (SQLDataAccess), Business (services for entities like Users, Products, Ingredients), and UI layers  
- **Database:** Logical model designed to support entities (e.g., Product, Ingredient, User, Sale, Purchase) with detailed relationships and attributes  
- **Interface:** Designed using Figma for prototyping, implemented with responsive and intuitive interfaces for both administrators and employees  
- **Services:** Specific services (e.g., UserService, ProductService, IngredientService) for handling business logic and data interactions  

---

## Testing & Results

- Validation of functional requirements, such as user authentication, product creation, and inventory management  
- Usability testing to ensure intuitive interfaces for employees and administrators  
- Performance evaluation focused on system response time and data consistency  
- Metrics included production rate (pizzas/hour), rejection rate (<2%), and average assembly time (<5 minutes per pizza)  
- Feedback from users confirmed ease of use and effective training for new employees
